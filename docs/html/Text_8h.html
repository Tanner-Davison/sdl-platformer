Detailed Description

SDL2-based text rendering component for displaying styled text.

This file provides a Text class that wraps SDL2_ttf functionality for rendering text with customizable colors, positions, and font sizes.

See also
SDL2 documentation at https://wiki.libsdl.org/
SDL_ttf documentation for font rendering details

Example usage:

Text myText("Hello World", 100, 50, 32);
myText.Render(windowSurface);
